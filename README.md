# DNF一键奔跑脚本

1.需要安装罗技GHub（百度自行下载），使用时需要打开

2.一键解压，不要移出文件夹任何文件

3.on.mp3和off.mp3可以根据自己喜欢替换

4.F1控制开关，打开会播放on.mp3，关闭会播放off.mp3

5.启动软件后会在右边状态栏显示图片，蓝色为开启状态，灰色为挂起或暂停状态

# V2版本
1.同上个版本

2.现在采用替换键位的方式进行辅助，更加灵敏，能够瞬启瞬停

3.需要设置物理按键和游戏按键

4.物理按键指实际操控的按键，即原来的左右箭头

5.游戏按键即游戏中设置的左右按键，建议设置成其他按键，设置成与物理按键相同可能做不到瞬启瞬停
